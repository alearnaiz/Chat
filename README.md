Chat
====
Implementación de un Chat. Para que la sincronización sea continua, cada medio segundo se llama al servidor para ver si hay conversaciones nuevas. 

Tecnologías
===========
* Backend: `Node.js`
* Frontend: `jQuery y HTML5`
